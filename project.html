TITLE: Home Automation using IOT

DESCRIPTION: The project mainly looks forward to connect the devices in our home to internet and can be controlled using a mobile application using Google Firebase as web server. This simple, yet effective project can solve a problem by controlling the LED via your smartphone over WiFi. Controlling over a web server means that it can be controlled from anywhere in the world! Having an internet connection is not a problem as most of the people already have it for their daily use.
